DEMO for npm module express-responsive-images

bird.jpg has a dimension of 4000x2000px and is stored in /public/images folder.

After the first visit to this page, the /public/images-cache directory was created. Resize the window, reload the page and observe the cache folder. Depending on the configuration, smaller images may be cached.

<img src="/images/bird.jpg">

This copy of bird.jpg is scaled to the window width and is cached in /public/images-cache/...

<img src="/images/bird.jpg?w=200">

This copy of bird.jpg is scaled to 200px in width and is cached in /public/images-cache/200/bird.jpg

<img src="/images/bird.jpg?w=400">

This copy of bird.jpg is scaled to 400px in width and is cached in /public/images-cache/400/bird.jpg
